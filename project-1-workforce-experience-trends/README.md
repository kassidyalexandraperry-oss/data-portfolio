# Project 1: Workforce Experience Trends (Pflugerville ISD)

## Objective
Analyze publicly available workforce data to identify teacher experience distribution patterns and potential retention risks.

## Data Sources
- NCES District Directory Summary (2024–25)
- Texas Education Agency TAPR (2023–24)

## Methods
- Descriptive statistics
- Data cleaning and normalization
- Basic data visualization in Excel

## Key Insight (Preliminary)
The largest share of teachers in Pflugerville ISD fall within the 1–5 year experience range, suggesting potential mid-career retention considerations.

## Limitations
- Data sources reflect different school years
- Analysis is limited to publicly available, aggregated data
